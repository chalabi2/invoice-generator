# invoicegenerator.xyz

## Dev

Install dependencies:

```bash
bun install
```

Run the dev server:

```bash
bun run dev
```

## Build

```bash
bun run build
```

## Cloudflare Pages

Build output: `dist`

```bash
bun run build
PROJECT_NAME=<project-name> bun run deploy
```
